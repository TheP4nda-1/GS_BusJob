## Was ist der BusJob?
Der BusJob ist ein ESX-basierter Job, bei dem Spieler als Busfahrer arbeiten
und feste Routen mit mehreren Haltestellen abfahren.

## Features
- Mehrere Routen
- Haltestellen-System
- Bezahlung pro Halt
- Abschlussbonus
- Navigations-Blips
- VollstÃ¤ndig konfigurierbar

## Voraussetzungen
- ESX Legacy
- Empfohlen: OneSync
- Funktionierendes Job-System
